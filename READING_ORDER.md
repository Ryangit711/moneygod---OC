## How to Read
1. Clone the repo (if you haven’t already):
   ```bash
   git clone git@github.com:Ryangit711/moenygod.git
   cd moenygod
   ```
2. Open `README.md` to see the dependency map (visual cross‑links).
3. Follow the numbered list above, opening each file in your editor (or `cat`/`less` in the terminal).
4. After reading or editing a file, run `obsync` (or `./scripts/git-auto-push.sh`) to commit and push the change so your progress is always reflected in GitHub.
5. If you want to jump to a specific topic, use the internal wiki‑style links (e.g., `[money](03-money-as-debt-modern-system.md)`) – the link‑validation hook ensures they stay valid.

Happy reading, and feel free to ask questions as you go—I’ll keep the wiki updated in the repo.
